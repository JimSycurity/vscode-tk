# Build rich table search and saved filter workspace

Add a richer table or webview mode for ticket scanning beyond the MVP TreeView, with multi-column sorting, stacked filters, saved filters, and body/full-text search.

## Acceptance Criteria

Users can switch to a table/webview ticket workspace; sortable columns cover id, title, status, priority, type, assignee, tags, project, updated time, and relationship counts; filters can be combined and saved; full-text search includes title, body, notes, acceptance criteria, tags, assignee, and external refs; README documents the relationship between TreeView and rich table modes.
